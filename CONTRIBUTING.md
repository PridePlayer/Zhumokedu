# 如何提交更改
Fork一个，修改，然后提交Pull request。在Pull request中，你最好写明白你修改的地方和原因及目的。通常PR会在一周之内回复。

# 如何报告错误
通过issue报告错误，请写明白发生的地方、预期结果、实际结果。如果有附图就更好了。会在一周之内回复。

# 如何请求增强
有些东西是你需要在项目中添加的，但是并不是错误问题一类致命的。你可以在issue中提出，说明需要的东西。会在一周之内回复。
